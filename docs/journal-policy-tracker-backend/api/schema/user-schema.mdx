## User Schema

```javascript
User: {
  id: ID;
  fullName: String;
  username: String;
  email: String;
  role: enum: {
    ADMIN: 'ADMIN';
    MODERATOR: 'MODERATOR';
    USER: 'USER';
  }
  createdAt: String;
  updatedAt: String;
}
```
