### `getCurrentUser` {#getCurrentUser}

<a className='response' href='../otherFolder/doc4.md'>
  ⮩ User
</a>

Returns a `User` object containing the data of the user who is currently logged.

<Tabs>
  <TabItem value='operation' label='Operation' default>

```graphql
query GetCurrentUser {
  getCurrentUser {
    id
    fullName
    username
    email
    role
    createdAt
    updatedAt
  }
}
```

  </TabItem>

#### `limitValue` ⟹ Int

#### `currentPageNumber` ⟹ Int

  ⮩ Paginated User
</a>

<Tabs>
  <TabItem value='operation' label='Operation' default>

```graphql
query GetAllUsers($currentPageNumber: Int!, $limitValue: Int!) {
  getAllUsers(currentPageNumber: $currentPageNumber, limitValue: $limitValue) {
    users {
      id
      fullName
      username
      email
      role
      createdAt
      updatedAt
    }
    totalUsers
  }
}
```
