### `getAllUsers(currentPageNumber, limitValue!)` {#getAllUsers}

Returns a paginated user object.

Returns an array of all the users who are currently registered in the database with pagination.

The following values are needed for the purpose of pagination:

#### `limitValue: Int!`

Number of Users that we want to display in a single page.

#### `currentPageNumber: Int!`

The page number of the page that is currently being displayed on the front-end.
